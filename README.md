# Excel-Project
Excel dashboard analyzing sports betting patterns, spending habits, addiction levels, and socioeconomic impacts across different demographic groups.


Project Overview

This project explores betting behavior among 2,000 individuals across multiple regions. Using Excel for data cleaning, analysis, and dashboard development, the study examines how betting habits affect personal finances, work-life balance, and relationships.

The analysis focuses on identifying behavioral patterns, spending trends, addiction indicators, and financial risks associated with sports betting.

Key Insights
Over 60% of participants reported having financial debt linked to betting activities.
Participants spend an average of approximately 19% of their monthly income on betting.
The average addiction score is 5.5 out of 10, indicating moderate gambling dependency across the sample.
Bet9ja emerged as the most frequently used betting platform in the dataset.
The analysis reveals clear relationships between betting frequency, addiction levels, and financial outcomes.
